# Youtube History

A React web application that isolates major YouTube creators (5 million+ subscribers) and presents their earliest videos in their "rawest" format. Users can explore the evolution of these creators by comparing their old content with newer videos, highlighting how they've changed as they've grown in popularity.

## Features

- Explore page showcasing major creators
- Contrast view to compare earliest vs. recent videos
- Algorithmic selection of creators meeting criteria (mock implementation)
- Responsive design for various screen sizes
- Built with React and Vite for fast development

## How to Run

1. Install dependencies: `yarn install`
2. Start the development server: `yarn dev`
3. Open http://localhost:5173 in your browser

## Technologies Used

- React 18
- Vite (build tool)
- CSS3
- JavaScript (ES6+)
